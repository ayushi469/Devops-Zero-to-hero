# Bash for DevOps

This repository documents my journey of learning Bash Shell Scripting for DevOps.

The goal was not only to understand Bash syntax but also to build automation scripts similar to those used by DevOps engineers.

## Topics Covered

- Shell Basics
- Variables
- User Input
- Command Line Arguments
- Conditional Statements
- Loops
- Functions
- Exit Status
- Pipes
- Input/Output Redirection
- grep
- awk
- sed

## Projects

- Server Health Checker
- Process Monitor
- Log Analyzer
- Disk Usage Report
- Log Backup Automation
- Deploy Validator
- Configuration Updater

## Skills Demonstrated

- Linux Automation
- File Handling
- Log Processing
- Text Manipulation
- Shell Scripting
- Bash Functions
